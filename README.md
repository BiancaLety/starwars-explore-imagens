# 🌌 Galeria de Ativos - STAR WARS EXPLORE

Este repositório contém o catálogo de imagens e recursos visuais utilizados no desenvolvimento do projeto **Star Wars Explore**. O objetivo é centralizar os arquivos para facilitar o consumo via URL direta ou submódulo.

## 📁 Estrutura de Pastas

* `/personagens`: Fotos e ilustrações dos personagens principais.

## 🔗 Como utilizar as imagens

Para exibir as imagens deste repositório em seu projeto HTML/React, utilize o link do **Raw Content** do GitHub. 

### Exemplo de URL:
`https://raw.githubusercontent.com/[Seu-Usuario]/[Nome-do-Repositorio]/main/personagens/exemplo.png`

### Exemplo de Uso no HTML:
```html
<img src="[https://raw.githubusercontent.com/usuario/repo/main/pasta/imagem.png](https://raw.githubusercontent.com/usuario/repo/main/pasta/imagem.png)" alt="Descrição">
